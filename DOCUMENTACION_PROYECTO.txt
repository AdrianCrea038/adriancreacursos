                          1. RESUMEN EJECUTIVO
================================================================================

Este es un sitio web profesional tipo landing page para un diseñador gráfico 
con 16+ años de experiencia. Incluye:

- Portafolio filtrable por categorías
- Sistema de cursos con venta por WhatsApp
- Panel de administración completo
- Diseño gamer/neón con efectos visuales
- 100% responsive (móvil, tablet, desktop)
- Funcionamiento offline total

El cliente puede gestionar TODO el contenido sin tocar código.

                      6. FUNCIONALIDADES DETALLADAS
================================================================================

6.1 SISTEMA DE BASE DE DATOS (database.js)
-------------------------------------------
Estructura en localStorage:

{
  "ar_projects": [
    { id, title, category, image }
  ],
  "ar_courses": [
    { id, title, description, price, image }
  ],
  "ar_users": [
    { id, username, password, role }
  ],
  "ar_config": {
    whatsapp: "521234567890",
    email: "contacto@adrianreyescrea.com"
  },
  "ar_db_version": 2
}

Características:
- Migración automática de datos viejos
- Fallback en 3 capas para imágenes
- Reset completo con db.resetAll()
- 100% offline sin dependencias

6.2 SISTEMA DE AUTENTICACIÓN (auth.js)
---------------------------------------
Funciones:
- auth.login(user, pass) - Valida y redirige según rol
- auth.logout() - Cierra sesión
- auth.checkAuth(requireMaster) - Verifica sesión y rol
- auth.getCurrentUser() - Retorna usuario actual

Credenciales por defecto:
Master: admin / admin123
Usuario: usuario1 / 1234

6.3 PANEL DE ADMINISTRACIÓN (admin.html + admin.js)
----------------------------------------------------
Secciones:

1. PROYECTOS
   - Formulario: título, categoría, URL imagen
   - Tabla con lista de proyectos
   - Botón eliminar por proyecto

2. CURSOS
   - Formulario: título, descripción, precio, imagen
   - Tabla con lista de cursos
   - Botón eliminar por curso

3. USUARIOS
   - Formulario: nombre, contraseña, rol
   - Tabla con lista de usuarios
   - No permite eliminarse a sí mismo

4. CONFIGURACIÓN
   - Editar WhatsApp y email
   - Estadísticas en tiempo real
   - Botón de reset (zona de peligro)

6.4 FORMULARIO DE CONTACTO (index.html + main.js)
--------------------------------------------------
Flujo:
1. Usuario llena formulario
2. Click en "Enviar por WhatsApp"
3. Se abre MODAL con vista previa
4. Usuario confirma
5. Abre WhatsApp con mensaje pre-llenado
6. Usuario envía desde su WhatsApp

Ventajas:
- No sale de la página hasta confirmar
- Revisión del mensaje antes de enviar
- Formato profesional con negritas
- Número configurable desde admin

                      8. CORRECCIONES REALIZADAS
================================================================================

#   PROBLEMA                          SOLUCIÓN
--  --------------------------------  --------------------------------------
1.  Login no redirigía a admin        Validación de rol + redirección
2.  Textos glitch difíciles de leer   Removido glitch en admin
3.  Admin incompleto                  Agregadas todas las secciones
4.  Sin ver contraseña                Toggle eye icon en login
5.  Contacto salía de página          Modal de confirmación
6.  Imágenes placeholder rotas        Migración a SVG offline
7.  Loops infinitos en imágenes       Flag dataset + onerror=null
8.  style.css corrupto                Reemplazado completo
9.  localStorage datos viejos         Versionado + migración auto
10. Sin reset de datos                Botón en admin (zona peligro)
